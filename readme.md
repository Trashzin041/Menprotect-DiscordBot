# Menprotect Discord Bot
## Installation
Install [Node.js](https://nodejs.org/dist/v16.17.1/)<br>
Goto Menprotect-DiscordBot folder and run the following command via shell/cmd<br>

Windows/Linux:
```
npm install
```

Edit config.json:
```
{
    "prefix": "!",
    "token": "Your-Token-Here",
	"channel": "ChannelID",
	"owner": "ClientID",
	"whitelist": [
		"ClientID", 
		"ClientID"
	]
}
```

Now run your bot:
```
node bot
```
